# 📄 Resume Keyword Matcher

## 🔍 Description
This project helps users check how well their resume matches a job description.

It compares both texts and provides:
- Match percentage
- Common keywords

## 🚀 Features
- Simple and easy to use interface
- Fast text comparison
- Useful for job preparation

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/resume-matcher.git

2. Navigate to folder:
   cd resume-matcher

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

## 📌 Usage
- Paste your resume
- Paste job description
- Click "Check Match"
- View results
